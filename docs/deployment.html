Operations / Deployment

Deployment

Current runtime deployment is Vercel for app services and GitHub Pages for docs.

Frontend Deployment (Vercel)

  • Config file: frontend/vercel.json
  • Rewrite rules:
    • /ai/(.*) → /$1
    • /ai → /
  • Build command from package scripts: npm run build.

Backend Deployment (Vercel Serverless)

  • Config file: backend/vercel.json
  • Entrypoint: backend/api/index.py exporting FastAPI app.
  • Route: all paths rewrite to api/index.py.

Release Checklist

  • Frontend build passes (npm run build in frontend).
  • Backend imports and startup path pass (uvicorn app.main:app ... in backend).
  • Critical routes return expected status: /health, /api/evaluate, /api/breathwork/protocols.
  • Environment values set in hosting dashboard.
  • Docs workflow green after push to default branch.
