MediaPipe Processing Path

flowchart LR
  CAM[UserCameraPanel video frame] --> LAND[usePoseLandmarker detectForVideo]
  LAND --> SMOOTH[Temporal smoothing in hook]
  SMOOTH --> STATE[landmarks state in App.tsx]
  STATE --> FRAME[computeFraming]
  STATE --> POST[/api/evaluate payload]
  POST --> RESP[AlignmentResponse]
  RESP --> UI[FeedbackPanel and overlays]

Frontend Session State Machine

stateDiagram-v2
  [*] --> welcome
  welcome --> disclaimer : continue
  welcome --> landing : profile exists
  disclaimer --> health_check
  health_check --> session_briefing
  session_briefing --> landing
  landing --> intro
  landing --> breathwork_session
  intro --> framing
  framing --> evaluating
  evaluating --> results
  results --> intro : next pose
  results --> sequence_complete : end sequence
  sequence_complete --> landing
  breathwork_session --> landing

Evaluate Request Lifecycle

sequenceDiagram
  participant FE as Frontend
  participant API as FastAPI Evaluate
  participant EV as AlignmentEvaluator
  participant PR as Prompt Builder
  participant GR as GroqClient

  FE->>API: POST EvaluateRequest payload
  API->>EV: evaluate landmarks
  EV->>PR: build_pose_context_block
  EV->>GR: evaluate_alignment with biomech summary
  GR-->>EV: JSON response
  EV-->>API: GeminiAlignmentResponse
  API-->>FE: alignment JSON + credit fields
