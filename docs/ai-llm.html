Prompt Design Principles (Verified)

  • Grounding-first: output must map to provided biomechanical evidence.
  • No fabricated praise/corrections for out-of-scope metrics.
  • Pose-specific violations are prioritized if available.
  • User-facing text should avoid explicit numeric measurements.
  • When deterministic score is high (>= 90), tone should be mostly positive with minimal correction.

Assistant Behavior

AssistantService enforces a scoped Madhu persona for yoga/breathwork and returns strict JSON with optional product suggestions.

  • Default format: {"reply":"...","suggestion":null}
  • Suggestion types: pose or breathwork
  • Context window: last 10 conversation messages
  • Fallback message is used on rate-limit/quota or parsing issues

TTS/Translation Notes

  • English (en-IN) uses pinned Neural2 voice names for consistency.
  • Non-English path translates from English first, while preserving Sanskrit pose names using placeholder protection.
  • In-memory warm-instance caches used for translation and audio bytes.
