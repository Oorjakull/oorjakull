Auth and CORS

  • Auth dependency: optional and required user resolvers in app/core/auth.py.
  • Google OAuth credential verification endpoint: /api/auth/google.
  • CORS loaded from CORS_ORIGINS with defaults for localhost, capacitor, and oorjakull domains.

Error Handling Notes

  • LLM failures fallback to safe placeholder correction text.
  • Credit deduction failure is non-fatal; evaluation response is still returned.
  • TTS/Translate upstream failures map to HTTP 502 with short diagnostics.
