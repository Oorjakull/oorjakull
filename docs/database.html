Credit System Logic

  • Enum profile_type: super_user, paid_user, free_user.
  • Trigger auto-creates default free_user profile with 20 credits on new user insert.
  • deduct_credit(p_user_id) RPC performs atomic decrement and returns JSON status.
  • NULL credits_remaining means unlimited user.
