Setup / Getting Started

Getting Started

Exact local setup commands from README and package/dependency files.

Prerequisites

  • Node.js + npm (version not pinned in repo files).
  • Python (version not pinned in repo files).
  • Android Studio only if building mobile with Capacitor.
SCAN NEEDED: Node/Python version constraints. Check .nvmrc, engines in package.json, pyproject.toml, or .python-version (not found in current scan).

Clone

git clone <REPO_URL>
cd Yoga_GenAI
SCAN NEEDED: Exact repository remote URL from git config/origin.

Frontend Setup

cd frontend
npm install
npm run dev

Default frontend dev port is 5173 from frontend/vite.config.ts.

Backend Setup

cd backend
python -m pip install -r requirements.txt
python -m uvicorn app.main:app --app-dir backend --reload --port 8000

Default backend port is 8000 from the README command.
